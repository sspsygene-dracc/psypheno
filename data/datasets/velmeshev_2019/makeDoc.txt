#!/bin/zsh
# Provenance for the velmeshev_2019 dataset.
#
# Paper:    Velmeshev et al., "Single-cell genomics identifies cell
#           type-specific molecular changes in autism."
#           Science 364(6441):685-689, 2019.
#           PMID 31097668, DOI 10.1126/science.aav8130.
#           First author: Dmitry Velmeshev.
#           Senior author: Arnold R. Kriegstein (UCSF).
#
# Source:   Supplementary Table S4 ("ASD_DEGs" sheet in the XLS):
#           ASD-vs-control DEGs per cell type from single-nucleus
#           RNA-seq of postmortem cortex (15 ASD + 16 control donors).
#           692 differential expression events (513 unique genes),
#           all with q < 0.05 and |log2FC| >= 0.14 (~10% change).
#
#           The XLS file was manually downloaded from the Science
#           article supplementary-materials page:
#               https://www.science.org/doi/10.1126/science.aav8130
#           Direct link (may require institutional access at first
#           visit; article itself is open-access):
#               https://www.science.org/doi/suppl/10.1126/science.aav8130/suppl_file/aav8130_table_s4.xls
#
# Raw data (snRNA-seq reads):
#   SRA accession PRJNA434002 (open access).
#
# Interactive browser:
#   https://autism.cells.ucsc.edu  (UCSC Genome Browser companion)
#
# Cadence: Fixed publication supplement; updates only on a Science
#          erratum. No re-download needed unless the supplement is
#          corrected.
#
# Sheets in the XLS (only ASD_DEGs ingested here):
#
#   ASD_DEGs (692 rows) — per-gene per-cell-type ASD-vs-CTL DEGs,
#     combined across PFC + ACC.  THIS IS THE INGESTED TABLE.
#
#   ASD_regional_DEGs (206 rows) — genes significant in PFC or ACC
#     but not both; separate fold-change + q-value columns for each
#     region.  Not ingested: adds complexity without clear benefit
#     for the per-gene portal view; available via raw download link
#     above if a follow-up table is warranted.
#
#   Bulk_deconvolution_DEGs (217 rows) — DEGs from bulk-tissue
#     RNA-seq deconvolution analysis; explicitly labelled log2(FC)
#     with both P value and FDR columns.  Not ingested: methodologi-
#     cally distinct from the snRNA-seq results and less directly
#     comparable to other datasets in the portal.
#
#   Individual_specific_DEGs (1071 rows) — per-patient directional
#     change counts (UP/DOWN, number of ASD patients showing signal).
#     Not ingested: no fold-change or q-value per row; patient IDs
#     are comma-separated strings, not suitable for gene-portal rows.
#
# Issue: https://github.com/sspsygene-dracc/psypheno/issues/198
